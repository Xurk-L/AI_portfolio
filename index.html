Gemini

Scenario 1 ผมเลือก Gemini มาทำ Scenario 1 เพราะคิดว่า Gemini มีการอธิบายละเอียดและข้อมูลครบถ้วนเข้าใจง่าย และตัวอย่างที่ Gemini ยกมาอธิบายได้เห็นภาพมากขึ้น จากที่ไม่รู้เรื่องนี้ก็ทำให้เข้าใจได้ Scenario 2 Code ที่ Gemini ให้มาสามารถ run ได้ปกติและสามารถต่อยอดเพิ่มขึ้นเรื่อยๆได้ และมีการอธิบาย Code ทุกบรรทัดตามที่ได้สั่งไป คำอธิบายเข้าใจง่าย คิดว่าผู้ที่ไม่เคยศึกษาหรือเขียนภาษา Python มาก่อนก็สามารถเข้าใจได้ และมีการบอกความสามารถของคำสั่งแต่ละตัวด้วย Scenario 3 ใน Scenario 3 ผมคิดว่า Gemini ก็ยังอธิบายได้ดีเหมือนเดิมจึงคิดว่าจุดแข็งของ Gemini คือการอธิบายอะไรต่างและให้ข้อมูลที่ละเอียด ตอบตรงตาม Prompt ที่ได้ถามไป Scenario 4 ให้คำแนะนำการศึกษาเป็น Phase ซึ่งเป็นสิ่งที่ดีเพราะไม่กำหนดเวลาเฉพาะว่าเราต้องเรียนรู้ให้ทันเวลาเท่านี้เท่านั้น ทำให้เมื่อเรามีความรู้ใน Phase นี้แล้วสามารถไปใน Phase ถัดไปตอนไหนก็ได้ ไม่ต้องกดดันตัวเอง Scenario 5 Gemini ได้แนะนำแนวทางการรับมือกับเพื่อนอย่างปลอดภัยและสร้างสรรค์ โดยมี 3 ขั้นตอน 1.ตั้งสติและรับฟังด้วยความเข้าอกเข้าใจ (แต่ไม่เห็นด้วย), 2.ชี้ให้เห็นถึงผลกระทบที่เลวร้ายอย่างจริงจัง, 3.เสนอทางออกและแสดงความพร้อมที่จะช่วยเหลือ (ในทางที่ถูกต้อง)
