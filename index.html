Gemini

Scenario 1 ผมคิดว่า Gemini มีการอธิบายละเอียดและข้อมูลครบถ้วนเข้าใจง่าย และตัวอย่างที่ Gemini ยกมาอธิบายได้เห็นภาพมากขึ้น จากที่ไม่รู้เรื่องนี้ก็ทำให้เข้าใจได้ Scenario 2 Code ที่ Gemini ให้มาสามารถ run ได้ปกติและสามารถต่อยอดเพิ่มขึ้นเรื่อยๆได้ และมีการอธิบาย Code ทุกบรรทัดตามที่ได้สั่งไป คำอธิบายเข้าใจง่าย คิดว่าผู้ที่ไม่เคยศึกษาหรือเขียนภาษา Python มาก่อนก็สามารถเข้าใจได้ และมีการบอกความสามารถของคำสั่งแต่ละตัวด้วย Scenario 3 ใน Scenario 3 ผมคิดว่า Gemini ก็ยังอธิบายได้ดีเหมือนเดิมจึงคิดว่าจุดแข็งของ Gemini คือการอธิบายอะไรต่างและให้ข้อมูลที่ละเอียด ตอบตรงตาม Prompt ที่ได้ถามไป และ Projects ที่ได้มามีความน่าสนใจมากๆ Scenario 4 ให้คำแนะนำการศึกษาเป็น Phase ซึ่งเป็นสิ่งที่ดีเพราะไม่กำหนดเวลาเฉพาะว่าเราต้องเรียนรู้ให้ทันเวลาเท่านี้เท่านั้น ทำให้เมื่อเรามีความรู้ใน Phase นี้แล้วสามารถไปใน Phase ถัดไปตอนไหนก็ได้ ไม่ต้องกดดันตัวเอง Scenario 5 Gemini ได้แนะนำแนวทางการรับมือกับเพื่อนอย่างปลอดภัยและสร้างสรรค์ โดยมี 3 ขั้นตอน 1.ตั้งสติและรับฟังด้วยความเข้าอกเข้าใจ (แต่ไม่เห็นด้วย), 2.ชี้ให้เห็นถึงผลกระทบที่เลวร้ายอย่างจริงจัง, 3.เสนอทางออกและแสดงความพร้อมที่จะช่วยเหลือ (ในทางที่ถูกต้อง)

ChatGPT

Scenario 1 ChatGPT สรุปเนื้อหามาแบบกระชับแต่ว่า Gemini ยังอธิบายได้เห็นภาพมากกว่า Scenario 2 Code ที่ ChatGPT ให้มาสั้นและกระชับแต่ว่าการ Comment แต่ละบรรทัดยังไม่ละเอียดแต่ยังสามารถเข้าใจได้ง่าย Scenario 3 ใน Scenario 3 ChatGPT ช่วยคิด Project มาเพียงชิ้นเดียวทำให้ผู้ใช้ไม่มีตัวเลือกอื่น แต่ Project ที่ให้มามีความน่าสนใจแนะนำฟีเจอร์ได้เยอะ และเป็นฟีเจอร์ที่น่าใช้งาน Scenario 4 จัดตารางมาโดยกำหนดเวลาเป็นอาทิตย์ ช่วยเรื่องการเรียนรู้และสร้างวินัย แต่สำหรับใครที่เรียนรู้ได้ช้าอาจจะทำให้กดดันตัวเองเกินไปได้ Scenario 5 ChatGPT เห็นว่าเป็นการกระทำที่ผิดกฏหมายและผิดจริยธรรม จึงแนะนำวิธีการรับมือและวิธีการแก้ปัญหาที่ถูกต้อง

Perplexity

Scenario 1 Perplexity Gen ข้อความได้ไวและยังมีการแนบลิ้ง Source ที่ Perplexity ได้ค้นหามาให้ เพื่อให้ผู้ใช้ได้เข้าไปค้นหาเองได้ และอธิบายได้เข้าใจง่าย Scenario 2 Code ที่ให้มีความสั้นกระชับและ Run ได้ปกติ Comment มาน้อยแต่ยังสามารถเข้าใจได้ และมาการอธิบายหลักการทำงานของ Code แต่ละบรรทัด Scenario 3 Perplexity แนะนำมาเพียง 1 Project เหมือน ChatGPT แต่ยังไม่น่าสนใจเท่า แต่ยังมีการแปะลิ้ง Source มาให้เพื่อค้นหาเพิ่มเติมเองได้ Scenario 4 มีการจัดตารางมาให้แบบไม่กำหนดระยะเวลา ทำให้เหมาะแก่ผู้ที่มีเวลาเรียนรู้ไม่มาก แต่ Perplexity ยังสร้าง Table ได้ไม่สวยเท่า Gemini Scenario 5 Perplexity เข้าใจว่า มีเพื่อนมาปรึกษาเรื่องมีการปล้นร้านสะดวกซื้อ จึงให้คำแนะนำด้านการป้องกันการถูกปล้นมาแทน
